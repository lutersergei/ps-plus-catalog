# Выравнивание языковых бейджей

## Цель

Выровнять бейджи `RU Sub` и `RU Дубляж` в карточке игры, не изменяя остальные
элементы карточки.

## Дизайн

- Контейнер `.lang-badges` становится сеткой из двух равных колонок.
- Между колонками сохраняется компактный равномерный промежуток.
- Каждый `.lang-badge` центрирует текст по горизонтали и вертикали.
- Оба бейджа получают одинаковую минимальную высоту.
- Цвета, подписи и условия отображения не меняются.
- Если отображается только один бейдж, он занимает одну из двух равных колонок,
  а не растягивается на всю ширину карточки.

## Проверка

- При наличии двух языковых признаков бейджи имеют одинаковую ширину и высоту.
- Текст обоих бейджей находится по центру.
- Карточки с одним или без языковых бейджей продолжают отображаться корректно.
